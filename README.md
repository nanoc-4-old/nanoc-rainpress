# nanoc-rainpress

This provides a [Rainpress](https://code.google.com/p/rainpress/) filter for [nanoc](http://nanoc.ws).

## Installation

`gem install nanoc-rainpress`

## Usage

```ruby
filter :rainpress
```

Parameters passed to this filter will be passed on to Rainpress.
